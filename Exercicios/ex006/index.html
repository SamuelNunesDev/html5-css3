Principais Formatações

Negrito ou Destaque

Podemos destacar um determinado texto usando negrito através da tag <b> (não semântica)

Podemos destacar um determinado texto usando negrito através da tag <strong> (semântica)

Itálico ou Ênfase

Podemos dar ênfase em um determinado texto utilizando itálico através da tag <i> (não semântica)

Podemos dar ênfase em um determinado texto utilizando itálico através da tag <em> (semântica)

Texto Sublinhado

Podemos sublinhar um texto utilizando a tag <u> (não semântica)

Podemos sublinhar um texto utilizando a tag <ins>(semântica)

Texto Excluído

Podemos utilizar a formatação de texto excluído para um texto que deve ser lido porém desconsiderado.

Texto Marcado

Podemos criar um texto marcado utilizando a tag <mark>

Podemos criar outro texto marcado utilizando a tag <mark>

Texto Grande e Texto Pequeno

O texto grande só é possível utilizar com as CSS, já o texto pequeno é possível utilizar a tag <small>

Texto sobrescrito e subscrito

Para utilizar um texto sobrescrito basta utilizar a tag <sup> Exemplo: x3=9

Para utilizar um texto subscrito basta utilizar a tag <sub> Exemplo: CO2
